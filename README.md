# Datos Primitivos

### Descripción

Este proyecto de consola muestra ejemplos de uso de los tipos de datos primitivos en Java:

* byte
* short
* int
* long
* float
* double
* char
* boolean

### Requisitos

* Java 17+ (recomendado)
* IDE como IntelliJ, Eclipse o VS Code
