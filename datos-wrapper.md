# Datos Wrapper

### **Descripción**

Este proyecto de consola muestra ejemplos de uso de los tipos de datos wrapper en Java:

* `Byte`
* `Short`
* `Integer`
* `Long`
* `Float`
* `Double`
* `Character`
* `Boolean`

### Requisitos

* Java 17+ (recomendado)
* IDE como IntelliJ, Eclipse o VS Code
