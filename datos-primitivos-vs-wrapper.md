# Datos Primitivos VS Wrapper

### Descripción

Este proyecto de consola muestra ejemplos de uso de los tipos de datos primitivos VS wrapper en Java:

Primitivo&#x20;                                        Wrapper Class\
byte&#x20;                                                       Byte\
short&#x20;                                                     Short\
int&#x20;                                                          Integer\
long&#x20;                                                       Long\
float&#x20;                                                      Float\
double&#x20;                                                  Double\
char&#x20;                                                      Character\
boolean&#x20;                                                Boolean

### Requisitos

* Java 17+ (recomendado)
* IDE como IntelliJ, Eclipse o VS Code
