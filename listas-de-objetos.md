# Listas de Objetos

### Descripción

Este proyecto de consola muestra ejemplos de uso de listas de Objetos en java:

Crear lista de objetos

Filtrado de listas

Ordenado de listas

Obtener objeto por propiedad

Buscar con función min y max

Filtrado con count

Lista inmutable

### Requisitos

* Java 17+ (recomendado)
* IDE como IntelliJ, Eclipse o VS Code
